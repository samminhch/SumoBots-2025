# SumoBots-2025

## Hardware Reference

This program is written for Nathan's [A1-Andromeda](https://github.com/nathan-jw/A1-Andromeda) repository

## Pinout Diagram

| Function            | Pin          |
| ------------------- | ------------ |
| Right motor in 1    | `D0`         |
| Right motor in 2    | `D1`         |
| Digital 2-6         | `D2` to `D6` |
| Left dist. sensor   | `D2`         |
| Center dist. sensor | `D3`         |
| Right dist. sensor  | `D4`         |
| Left motor in 2     | `D9`         |
| Left motor in 1     | `D10`        |
| Start-stop module   | `D11`        |
| Analog 3-7          | `A3` to `A7` |
